# Sprint 5 - Phoenix Document Workspace

Sprint 5 focuses on making business documents the core DocCraftTools product experience.

## Shipped
- Universal document workspace layout
- 10 document generators
- Live preview on every document page
- Template gallery: Modern, Minimal and Corporate
- Export toolbar: New, Duplicate, Print, Download PDF
- Workspace sidebar with next-step workflows
- Updated Documents hub

## Acceptance Criteria
- Every document uses the same layout and design system.
- Line items and totals update instantly.
- Templates can be changed without leaving the page.
- Mobile layout stacks cleanly without horizontal scrolling.
- Export actions remain consistent across all documents.
