# DocCraftTools QA Checklist

Use this before every release.

- Navigation works on desktop and mobile.
- Forms have labels and keyboard focus states.
- Tool preview updates after field changes.
- Export buttons remain visible and readable.
- Pages use the shared header, footer and design tokens.
- No business tool is added unless it supports the business toolkit strategy.
- Mobile layout has no horizontal scrolling.
- Related tools are included on every tool page.
