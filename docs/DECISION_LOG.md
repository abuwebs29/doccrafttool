# DocCraftTools Decision Log

## Decision 001 — Keep DocCraftTools as the brand
Reason: The name is broad enough for documents, calculators, HR, PDF and AI tools while preserving the existing domain.

## Decision 002 — Focus on business-related tools only
Reason: Focus improves brand trust, topical authority, SEO and user experience.

## Decision 003 — Build a design system before adding more tools
Reason: A shared system makes every future page faster to build and easier to maintain.
