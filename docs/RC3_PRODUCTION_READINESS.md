# RC3 Production Readiness

This release focuses on usable starter templates, reliable routing, search accessibility and deployment readiness.

## Deployment
Upload the contents of `doccrafttool-main` as the website root. Do not upload the enclosing folder as an extra path level.

## Manual checks after deployment
1. Open one template from `/templates/` and verify that the matching generator is prefilled.
2. Edit a line item and verify that the preview updates.
3. Use Print and choose Save as PDF.
4. Test Ctrl/Cmd + K, arrow keys and Enter.
5. Test the contact and legal pages on mobile.
